Environmental requirements:
NVIDIA GPU, Python >= 3.7, PyTorch >= 1.3.1, CUDA >= 10.0

MakeDataset. Py: Contains code for making data sets, data preprocessing, and image enhancement
AmresNet.py: Contains the detailed implementation code of the AmresNet architecture, which is completed using the deep learning framework PyTorch
Train.py: Contains training logic, detailed training parameter Settings, and code related to test results (model performance)
